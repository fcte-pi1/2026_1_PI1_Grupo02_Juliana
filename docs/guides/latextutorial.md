# **2 - Usando LaTeX no VS Code**

## **Instalações**

- [MiKTeX](https://miktex.org/download)

- [Pearl](https://strawberryperl.com/) (Escolha a opção MSI)

Após as duas intalações, abra o MiKTeX Console, vá em Settings e em Package installation e marque a opção **"Always"**.

![miktex](https://res.cloudinary.com/dqeji5jup/image/upload/f_auto,q_auto/image11_b3bgwv)

No VS Code, vá em Extensões e pesquise "latex workshop", clique na extensão da imagem e depois em **"Install"**.

![extensions](https://res.cloudinary.com/dqeji5jup/image/upload/f_auto,q_auto/image12_srbqqf)

Agora você já pode visualizar e editar arquivos LaTeX no VS Code!

Para visualizar o PDF do LaTeX, utilize o explorer de arquivos do VS Code e vá até o arquivo "ralatorio.tex".

![arquivos](https://res.cloudinary.com/dqeji5jup/image/upload/f_auto,q_auto/image15_sdm8o6)

Clique nos seguintes arquivos:

```
 docs > relatorio > relatorio.tex
```

Com **"relatorio.tex"** aberto, clique no "play verde" e aguarde o PDF ser feito.

![pdf](https://res.cloudinary.com/dqeji5jup/image/upload/f_auto,q_auto/image19_eukpfv)

Ao final do processo o arquivo **"relatorio.pdf"** será criado, abra ele para visualizar o pdf. As alterações feitas nos arquivos de LaTeX aparecerão automaticamente no PDF.
