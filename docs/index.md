# Bem-vindo ao Projeto 2026.1 PI1 - Grupo 02

Este é o repositório do Projeto Integrador I (PI1) do Grupo 02, orientado pela professora Juliana.

## Sobre o projeto

O trabalho está organizado por áreas:
- `docs/`: documentação e relatório
- `hw/`: hardware (esquemáticos, PCB, CAD)
- `mec/`: componentes mecânicos
- `src/`: código-fonte (backend, firmware, frontend)

## Guia rápido

```bash
git clone https://github.com/fcte-pi1/2026_1_PI1_Grupo02_Juliana.git
cd 2026_1_PI1_Grupo02_Juliana
```
